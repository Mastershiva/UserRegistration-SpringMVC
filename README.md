# User Registration - Spring MVC

## 📌 Overview
A responsive User Registration web application built using Spring MVC, JSP, HTML, CSS, JavaScript, Maven, MySQL, and Apache Tomcat.

## 🚀 Features
- User Registration Form
- Responsive User Interface
- Spring MVC Architecture
- MySQL Database Integration
- Form Validation
- Success Confirmation Page

## 🛠️ Technologies Used
- Java 8
- Spring MVC
- Maven
- JSP
- HTML5
- CSS3
- JavaScript
- MySQL
- Apache Tomcat 9
- Eclipse IDE

## 📂 Project Structure
```
src/
 ├── main/
 │   ├── java/
 │   ├── webapp/
 │   └── resources/
```

## ▶️ How to Run
1. Clone the repository.
2. Import as a Maven Project in Eclipse.
3. Configure MySQL credentials.
4. Run on Apache Tomcat 9.
5. Open:
```
http://localhost:8080/UserRegistration/
```
